# Installer j2ee-CV

Cette application permet de créer une liste de CV

##require

- java
- nodejs
- yarn ou NPM

cloner le projet: [https://github.com/KILLERMAX91/j2ee-CV.git](https://github.com/KILLERMAX91/j2ee-CV.git)

### installations des bibliothéques js css
	
##### installation avec yarn
	
```
cd WebContent
```
	
```
yarn install
```
	
```
yarn run bower update
```
	
	
##### installation avec NPM

```
cd WebContent
```

```
npm install
```

```
npm run bower update
```
